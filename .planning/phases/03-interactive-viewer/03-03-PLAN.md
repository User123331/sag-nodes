   - Compute `canvasWidth = windowWidth - (isPanelOpen ? 320 : 0)`

4. **Consolidate expand logic**: The expand handler should live in one place. Best approach: create an `expandNode` function in `engineSlice` (or a separate action) that both GraphCanvas and DetailPanel can call. This function:
   - Reads `engine` from store
   - Sets `expandingMbid` and `isExpanding`
   - Calls `engine.expand(mbid)`
   - On success: calls `addExpansion(result.value, expandingNode)`, calls `triggerReheat()`
   - Handles truncation toast and warnings
   - Resets `expandingMbid` and `isExpanding`

   Add to engineSlice:
   ```typescript
   performExpand: async (node: ForceNode) => {
     const { engine } = get();
     if (!engine) return;
     const { setExpandingMbid, triggerReheat, addExpansion } = get() as StoreState;
     setExpandingMbid(node.mbid);
     set({ isExpanding: true });
     const result = await engine.expand(node.mbid);
     if (result.ok) {
       addExpansion(result.value, node);
       triggerReheat();
       if (result.value.truncated) {
         toast(`Node limit reached (${result.value.nodeCount}/150). Graph is at maximum size.`, { duration: 5000 });
       }
       result.value.warnings.forEach(w => toast.error(`${w.provider}: ${w.error}`, { duration: 4000 }));
     } else {
       toast.error('Expansion failed. Try again.', { duration: 4000 });
     }
     setExpandingMbid(null);
     set({ isExpanding: false });
   }
   ```

   Then both GraphCanvas double-click and DetailPanel Expand button call `performExpand(node)`.
